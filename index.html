IOT를 이용한 스마트홈


이 시스템은 IOT를 이용해 집 안의 환경을 편리하게 제어할 수 있도록 도와주는 시스템이며, 독거노인이나 혼자 사는 사람들에 경우 보호자들이 그분들의 안전을 걱정하게 되는데 사람이 사는데 꼭 필요한 정수기 & 화장실에 센서를 이용해 움직임을 감지해 사용자의 안전을 확인할 수 있도록 만든 스마트홈 시스템 입니다.

개발인원/기간 : 2명/10일

나의역할
기획: 아이디어 구성 및 개발기획 주도
Arduino: 화재감지센서를 통해 값이 읽힌다면 MP3 플레이어 쉴드를 이용해 음성을 출력, 빗물감지센서를 이용해 빗물이 감지된다면 창문이 닫힘, 가스 감지센서를 이용해 가스가 감지 되었을경우 창문이 개방,Arduino orage board의 WIFI 통신을 이용해 초음파센서에 값이 감지 되었을 경우 pushbullet 을 통해 사용자의 핸드폰에 알람

VR 기반 지능형 가상면접 시스템


현재 지속적인 신규 채용 감소로 인해 취업난이 극심화되고 이로 인해 취준생의 면접 준비 부담이 증가되고 있다고 합니다. 이러한 부담감이 해소되고 취업을 돕기 위해 현재 많은 취업 프로그램들이 진행되고 있지만, 시간이 고정적이며 참여도가 낮고 인적자원 비용이 증가 된다는 단점을 가지고 있습니다. 그래서 저희는 이러한 단점을 해소 할 수 있는 VR을 이용한 가상 모의면접 시스템을 개발하게 되었습니다.

개발인원/기간 : 11명/7달

나의역할
기획: 아이디어 구성 및 개발기획 주도
Local Team Leader: Naver 음성합성 API TTS(Text to Speech)를 이용해 사용자에게 질문을 들려주기 위해 텍스트를 파일입출력을 통해 저장 및 MP3파일로 생성
GUI ProGram 구현을 위해 기획 및 Java FX를 이용해 구현
질문을 받기 위해 Server에서 보낸 Data 파싱
각 모듈이 동시에 돌아가기 위해 다중 Thread 사용 및 모듈 통합 안정화
